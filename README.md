# persiliao-tsconfig

> Basic tsconfig.json configuration for typescript

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D persiliao-tsconfig
```

## Usage

In `tsconfig.json`:

```json
{
  "extends": "persiliao-tsconfig/base.json"
}
```
